Miután kész az alkamazásunk fel kell mennünk herokuoldara

Ezen felül meg kellene nyitni egy másik ablakot

Itt a dokumentáció

Kikeresni mi is az keretrenszer amit használunk majd a getStartjára rányomni

Ha a node.js akkor feltelepíteni az adott programot.

leelemörizni a megfelelő verzió számokat

node -- versio

(a node itt csak példa)

Az adott projekt mappűában létre kell hozni EGY FORMÁTUM NÉLKÜLI filt "Procfile" néven

touch Procfile

Majd belemenni atommal és beleírni: web: node app.js (habár az oldal nállam azt írja, hogy: web: npm start)

Ezek után a hiperbe írjuk a következőt

git init

Hozzá kell adnunk a git-hez amit használunk.

git add .

Hogy vissza lehessen verziószám:

git commit -m "Firs commit"

Létre kell hozni egy új "alkamazást" Hiroku-n

heroku create

fel kell (tólni) tenni a LOCKAL verziót

git push heroku master

Vizsgálat.

heroku logs --tail
