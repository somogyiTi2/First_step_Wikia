A feladat egy LDAP-pal összekötött, különböző jogosultásgokat kezelő,naplozó program volt. Adatbázisban szinten, egy felhasználó táblára volt szükségem, egy jogosultság kezelő tábla, egy felhasználói tábla egy napló tábla (ebbe kerülnek a bejegyzések), és egy kategória tábla (hogy dinamikusan lehessen változtatni a kategória neveket).

használt technológiák:

HTML:5,CSS,PHP, JavaSricpt, jQuery, Bootstrap

(A képek adatvédelmi szempont miatt ilyen felbontásuak)
